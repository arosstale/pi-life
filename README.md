# pi-life 🧬

Conway's Game of Life for [pi](https://github.com/nicobailon/pi-mono).

## Install
```bash
npm install -g pi-life
```

## Play
```
/life              Random soup
/life glider       Single glider
/life gosper        Gosper glider gun
/life pulsar       Period-3 oscillator
/life acorn        Acorn methuselah
/life rpentomino   R-pentomino
/life spaceship    LWSS
```

## Features
- Half-block rendering (2x vertical resolution)
- Draw mode (D) — place cells and stamp patterns
- 6 color themes, speed control, single-step mode
- Born/alive/dying/dead cell coloring
- 80×30 toroidal grid

Controls: SPACE=pause N=step +-=speed D=draw R=random C=clear T=theme Q=quit
